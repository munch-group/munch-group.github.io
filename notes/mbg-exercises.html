mbg-exercises .sh/ps1 double-click script

Windows: Put commands In a .ps1 or .bat file like with Jakobs server so Windows students can just DoubleClick the file.

Similar bash script on Mac : If you give your script a .command extension and perform a chmod 777 on it (do make it executable), it will open and run in the Terminal when double-clicked.

NB: Should just activate mbg-exercises environment and call python entry point from mbg-exercises-client package and pass all arguments to that

mbg-exercises-client package

Entry point callable like

mbg-exercises mbg-exercises --verbose # also clone repo and check that all is pushed and offer to do so otherwise mbg-exercises --devel

Should:

  1. Activate mbg-exercises environment
  2. Check for package updates
  3. Automatically update if necessary
  4. Reimport package after update to use new version
  5. Read registry listing using API requests library (see above)
  6. Read yml file mapping English to Danish course names (see above)
  7. Take user input through the menu interface
  8. Check that docker is running
  9. Check that no other exercise container is running
  10. Check that no other local jupyter is using a port
  11. Launch docker run as a process in a separate thread (like in slurm-jupyter)
  12. Read URL from the output and launch the browser like in (like in slurm-jupyter)
  13. Make sure the script cannot be shut down with keyboard interrupt or kill without cleaning up
  14. Report how much disk space images and containers currently use
  15. Ask to purge docker images and containers [Y]/n and do so
  16. Exit
